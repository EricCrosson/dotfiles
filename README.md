Configurations managed by [chezmoi](https://github.com/twpayne/chezmoi).

## Building GNU Emacs on MacOS

``` shell
brew install emacs-head@28 \
    --with-cocoa \
    --with-imagemagick \
    --with-no-frame-refocus \
    --with-pdumper \
    --with-xwidgets \
    --with-modern-icon-purple
```
